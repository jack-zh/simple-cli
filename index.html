    Simple-CLI

    Simple-CLI是一个利用YACC和LEX实现的简易的命令行实现程序，只是一个最简化的CLI工具，可以自由扩展。https://github.com/jack-zh/simple-cli

    一个简单的Yacc 与 Lex 快速入门 YACC/LEX和 Yacc 与 Lex 快速入门

    一些这方面的书籍推荐： lex与yacc 和 flex与bison

    Installation

    在编译之前，确认安装以下包，注：clang只是我习惯的编译器，当然，你可以选择gcc

        install readline  
    sudo apt-get install bison
    sudo apt-get install flex
    sudo apt-get install clang

    ./cli
